My HTML Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in a `li` tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in a `li` tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

Tables

Tables are great for displaying tabular data.

Other Concent Section Tags

Articles

Articles elements represent a self contained composition in a document, page, application, or site.
This can be a forum post, a magazine, a newspaper article, or other independent item of content.

Headers

Header element represents introductory content, may contain some heading elements, a logo, a search form, an author name, and other elements

navs

Nav element provides navigation links, common examples include menus, table of contents, and indexes

Footers

Footer element encloses information for its nearest ancestor sectioning content or sectioning root element.
examples include information about the author of the section, copyright data or links to related documents.
